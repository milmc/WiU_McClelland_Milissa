Project Name: WorldSurf

Project Description

This website is for my work done in the Full Sail University class Web Interface and Usability. In this project I had to stylize this page using the HTML5 and CSS3 that I learned in DWS1. This landing page houses links to my Usability Hub tests. This project will show my understanding of such things as utilizing a visual hierarchy, design patterns, an understanding of information architecture as well as many other interface design elements.
